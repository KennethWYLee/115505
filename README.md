# 社福企劃助手 Impact Planner

GitHub repository name: `115505`

第 115505 組系統已整理為可操作的靜態前端工作台，包含互動任務、狀態管理與本機瀏覽器資料保存。

## 系統能力

- 集中政府補助公告、申請條件、服務對象與截止日期，讓 NPO 快速篩選。
- 解析簡章與服務構想，產生含目標、預算、KPI 與依據引用的企劃草稿。
- 以 RAG 連結法規與公告資料，降低錯誤引用並保留官方依據。
- 整合捐款、活動簽到、個案日誌與 CSR 名單，支援後續營運管理。
