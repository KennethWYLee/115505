# 社福企劃助手 Impact Planner

GitHub repository name: `115505`

此公開展示版依據第 115505 組學生文件的系統目的、功能與預期成果重新設計，重點是讓版型、視覺語言與操作情境更貼近該系統的產品定位。

## 展示重點

- 主動掃描政府補助公告，集中分類落後且分散的補助資訊。
- 上傳簡章 PDF 並輸入構想後，由 AI 產生標準格式企劃書草稿。
- 以 RAG 連結法規向量庫，降低 AI 幻覺並保留官方事實依據。
- 整合捐款、活動簽到與個案日誌，長期擴展 CSR 公益媒合。
